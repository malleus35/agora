# Skeptic Overlay

## Use when
- verification matters more than novelty
- release confidence is low
- claims need evidence checks
- risk review is required

## Pair with
- doubt-list

## Anti-patterns
- treating confidence as evidence
- stopping at happy path review

## Expected artifact
- doubt list with verification actions and release posture
