# Agora

> Philosophers plan through dialectics.

A Claude Code plugin. Seven philosopher agents debate using Hegel's thesis-antithesis-synthesis structure. Socrates draws out consensus through questioning. Descartes verifies the result.

## Install

```bash
# From marketplace (once published)
/plugin install agora

# From Git
/plugin marketplace add malleus35/agora
/plugin install agora
```

## Usage

```bash
# Start a debate session
/agora "Build a REST API for task management"

# Or summon agents directly
@socrates "I want to build a communication tool between AI agents"
```

## Agents

| | Philosopher | Role | Core Method |
|---|---|---|---|
| 🏛️ | **Socrates** | CEO / Orchestrator | Maieutics — draws out consensus through questions |
| 🔺 | **Hegel** | Planner / PM | Dialectics — plans through thesis-antithesis-synthesis |
| 🔲 | **Wittgenstein** | FE / Interface | Language games — context determines paradigm |
| ⬡ | **Spinoza** | BE / Infrastructure | Geometric method — deduces implementation from axioms |
| △ | **Kant** | Designer | Critique of Judgment — judges beauty by universal principles |
| ⚡ | **Nietzsche** | Marketer | Revaluation — declares new narratives |
| 🔍 | **Descartes** | QA | Methodical doubt — passes only what is clear and distinct |

---

## Design Principles

**Philosophy is methodology, not decoration.** Each agent genuinely operates through its philosopher's mode of thinking.

**Tension creates consensus.** Disagreement between agents is a feature, not a bug.

**Socrates holds no answers.** If the orchestrator has predetermined answers, debate becomes theater.

**Descartes is last.** No output is marked complete without QA.

---

## About the Name

Agora was the open public square of Athens — where Socrates debated citizens, and merchants mingled with philosophers. This is where the agents meet.
