# /decide — Decision Workflow

Use this command when a real fork must be framed and judged.

## Purpose

Route to:
1. frame-the-decision
2. compare-options
3. decision-memo
4. minority-report if unresolved dissent remains

## Output artifact

```markdown
## Decision Package

### Decision Frame
- ...

### Option Comparison
- ...

### Recommendation
- ...

### Decision Memo
- ...

### Minority Report
- optional
```

## Examples

```bash
/decide "Should Agora remain plugin-first or separate a host-agnostic core now?"

/decide "Choose between a shared-spec overlay and repo-per-agent architecture"
```
