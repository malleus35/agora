# /minority-report — Preserve Dissent

Use this command when a decision has been made but important objections remain alive.

## Purpose

Capture:
- unresolved objections
- discarded alternatives
- why they were not chosen now
- revisit triggers
- downside if the minority view is correct

## Output artifact

```markdown
## Minority Report

### Decision Taken
- ...

### Unresolved Objections
- ...

### Discarded Alternatives
- ...

### Revisit Triggers
- ...

### Risk if Minority View Is Correct
- ...
```

## Example

```bash
/minority-report "We chose a plugin-first strategy, but preserve the strongest argument for a future repo split"
```
