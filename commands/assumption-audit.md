# /assumption-audit — Genealogy Workflow

Use this command when the team may be trapped inside inherited language, values, or categories.

## Purpose

Produce:
- explicit assumptions
- hidden assumptions
- inherited categories
- value conflicts
- a what-if-the-opposite-were-true probe

## Output artifact

```markdown
## Assumption Audit

### Explicit Assumptions
- ...

### Hidden Assumptions
- ...

### Inherited Categories
- ...

### Value Conflicts
- ...

### What if the Opposite Were True?
- ...
```

## Example

```bash
/assumption-audit "Audit the assumptions behind our current framing of Agora as a debate harness"
```
