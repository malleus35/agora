# /steelman — Strongest-Form Restatement

Use this command when disagreement is becoming shallow, adversarial, or distorted.

## Purpose

Produce a strongest-form restatement before critique.

## Output artifact

```markdown
## Steelman

### Target Position
- ...

### Strongest-Form Restatement
- ...

### What This Position Protects
- ...

### Fair Critique Boundary
- ...
```

## Examples

```bash
/steelman "Restate the strongest case for keeping Agora plugin-first before we criticize it"

/steelman "Steelman the policy argument against autonomous release agents"
```
