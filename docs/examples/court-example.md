# Court Review Example

```bash
/court-review "Review whether Agora v2 is ready for a public positioning update"
```

```markdown

- The repositioning improves adoption by framing Agora as an overlay rather than a replacement.

### Censor

### Sovereign Verdict
- Approve with conditions
- Conditions: publish philosophy map, examples, and host install docs before public push.
```
